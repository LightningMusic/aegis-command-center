📊 Productivity Module
Features

Add / Edit / Delete tasks

Categories (School, Personal, Projects)

Estimated time (minutes)

Due date

Completion toggle

Task filtering

Smart Priority Algorithm

Example logic:

urgency_score = max(0, 10 - days_until_due)
importance_score = importance * 2
time_weight = min(5, estimated_minutes / 60)

priority_score = urgency_score + importance_score + time_weight

This score determines:

Task sorting

“Work Now” suggestion

Dashboard high-priority count

Study Timer

Linked or standalone

Start / Pause / Stop

Stored in Study Sessions table

Updates status bar

Appears in dashboard panel

Uses QTimer (safe in GUI thread)

Optional later:

Deep Work Mode toggle

📁 Smart File Organizer
Core Features

Add monitored folders

Activate / deactivate monitors

Real-time file detection (watchdog)

Auto-sort rules:

By extension

By creation date

By file type folder

Duplicate Detection

SHA-256 hashing

Compare hashes against DB

If match:

Move to “Duplicates” folder OR

Log and prompt user

Runs in QThread to avoid freezing UI.

Rename Engine

Template rules:

{date}_{original}

{category}_{counter}

{ext}_{timestamp}

Action Log

Every file action logs:

Original path

New path

Timestamp

Action type

Undo feature:

Reverse last move using stored paths
